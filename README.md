# 📊 HR Data Analysis with Python

## 📌 Overview

This project performs **Exploratory Data Analysis (EDA)** on an HR employee dataset using Python. The analysis aims to uncover valuable insights related to employee demographics, salaries, departments, performance, hiring trends, and attrition. The project demonstrates how data analysis can help organizations make informed HR decisions.

---

## 🎯 Objectives

- Clean and preprocess the HR dataset
- Analyze employee distribution across departments
- Compare Remote and On-site employees
- Study salary distribution across departments and job roles
- Identify top-paying job titles
- Analyze employee performance ratings
- Examine hiring trends over the years
- Calculate department-wise attrition rates
- Discover relationships between salary, experience, and performance
- Generate business insights using data visualization

---
